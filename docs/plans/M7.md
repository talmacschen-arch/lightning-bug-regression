## 单一分支：`feat/m7-llm-generate-draft`

- [ ] **M7-prep（依赖）**

- [ ] **M7-1 backend endpoint** `POST /cases/generate-draft`（`backend/app/api/cases.py`）
  - **Bearer auth 必须**（**A**）—— `Depends(current_user)` 与 v1.17+ 对齐，不沿用 `/cases/*` 当前匿名姿态
  - request：`{description: str, category?: str | None}`；**`len(description) > 8 KB` → 413**（**B**）
  - response：`{yaml_draft: str, attempts: int, validation_errors_during_retry: list[str]}`
  - 实现：Anthropic SDK，`max_tokens=2000`（**B**），prompt caching 见 M7-2
  - 校验复用 `yaml_loader.parse + case_normalizer.normalize_case`（§14 R26），**禁** inline 校验
  - **retry 分两类**（**C**）：
    - schema-invalid（输出过不了 yaml_loader/normalize）→ retry ≤2 次，**上次的 validation error 串塞回下次 prompt**
    - Anthropic API 错（429 / 5xx / 网络 / 超时）→ **不重试 / 不进 retry quota**，按错误码上抛
  - **错误码细分**：
    - ANTHROPIC_API_KEY 缺 → **503**
    - Anthropic 5xx → **502**
    - Anthropic 429 → **429** 透传
    - 网络/超时 → **504**
    - description > 8KB → **413**
    - no Bearer → **401**
    - 2 次 schema retry 失败 → **200** + body `attempts=3 + validation_errors_during_retry`（业务态非错）
  - **可观测性**（**F**）：每次调用 `logger.info` 落 `prompt_tokens / completion_tokens / latency_ms / retry_count / validation_errors_count / model`；不持久化

- [ ] **M7-2 prompt module** 新建 `backend/app/api/llm_prompt.py`
  - System prompt 内容：
    - §4.1 YAML schema 简版
    - canonical 字段顺序（§17.6）
    - status_whitelist（按 category 注入，**`{{ALLOWED_CATEGORIES}}` placeholder fetch from `/admin/categories` at request time**，§14 R4b 不硬编码）
    - **(E-1)** 显式写"未指定 database 时填 `gpadmin`"（2026-05-24 新规范，不只靠 few-shot 推断）
    - **(E-2)** 显式写 §4.1.2 psql -c 铁律 —— non-tx-safe DDL（VACUUM / REINDEX CONCURRENTLY / CREATE DATABASE / ANALYZE 顶层）**绝禁** `kind: sql`，**必须** `kind: shell + cmd: psql -c '<DDL>'`
  - 3 个 hardcoded few-shot 例：
    - `lg-bug-0001-hashjoin-right-table`（BUG，简单 GUC + SELECT）
    - `lg-ext-pgvector-ivfflat-basic`（extension，CREATE EXTENSION + INDEX + plan_contains）
    - `lg-bug-0008-pax-toast-vacuum-analyze-crash`（cross-driver `kind: sql` + `kind: shell psql -c`，§4.1.2 实战）
  - **Prompt caching**（**G**）：schema + few-shot 共同前缀打 `cache_control: {"type": "ephemeral"}` marker；user `description` **不**缓存。注：5-min TTL，低流量场景命中率有限——优化定位而非承诺。
